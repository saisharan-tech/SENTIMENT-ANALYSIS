## PROJECT DESCRIPTION

This project focuses on performing sentiment analysis on Amazon product reviews using Natural Language Processing (NLP) techniques. The objective is to classify customer reviews into positive or negative sentiments and extract meaningful insights from textual data.

Customer reviews play a crucial role in understanding product performance and customer satisfaction. By applying sentiment analysis, businesses can automatically analyze large volumes of reviews and make data-driven decisions to improve products and services.

The dataset used in this project contains labeled Amazon product reviews. Each review is processed, cleaned, and transformed into a format suitable for machine learning models.

---

## FEATURES

- Text data preprocessing and cleaning  
- Removal of stopwords, punctuation, and noise  
- Tokenization and text normalization  
- Feature extraction using TF-IDF  
- Sentiment classification (Positive / Negative)

- Performance evaluation using accuracy and confusion matrix

## WORKING

The project is implemented in multiple stages:

### 1. Data Preprocessing
- Loaded raw Amazon review dataset  
- Removed unnecessary characters and noise  
- Converted text to lowercase  
- Removed stopwords  
- Applied tokenization  

### 2. Feature Engineering
- Converted text into numerical form using TF-IDF vectorization  

### 3. Model Implementation
- Applied Logistic Regression model for classification  
- Trained the model on processed data  

### 4. Evaluation
- Measured model performance using accuracy  
- Generated confusion matrix to analyze predictions  

### 5. Insights
- Identified distribution of positive and negative reviews  
- Analyzed patterns in customer feedback
